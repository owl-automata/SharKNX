# Pagina Settings (Impostazioni)

La pagina Settings è accessibile da qualsiasi schermata toccando l'**icona dell'ingranaggio** situata nell'angolo in alto a destra della barra dell'applicazione. Un'**icona di ricerca** nella barra superiore consente di saltare direttamente a una sezione cercandola per nome.

Le impostazioni sono raggruppate nelle seguenti sezioni:

| Sezione | Cosa controlla |
|---|---|
| **Discover** | Indirizzo e porta multicast, scansione di fallback unicast, timeout di rilevamento e connessione |
| **Project** | Opzione per caricare gli oggetti di comunicazione; numero di elementi visualizzati per pagina nell'albero del progetto |
| **Monitor** | Mantenimento dello schermo attivo (Keep-screen-on) durante il monitoraggio; dimensione massima del buffer dei telegrammi |
| **Scan** | Durata della scansione in modalità di programmazione e intervallo di ping (utilizzati nella scheda Management → Prog. Mode) |
| **Subscription** | Stato dell'abbonamento attuale, ripristino degli acquisti, annullamento dell'abbonamento, Informativa sulla Privacy, Termini di Servizio |
| **Language** | Lingua dell'interfaccia dell'app (Inglese, Tedesco, Francese, Spagnolo, Italiano) - ha effetto immediato |
| **Theme** | Tema Chiaro, Scuro o Allineato al sistema |
| **Send Feedback** | Lascia una recensione, segnala un bug, scrivici |

Per la descrizione completa di ogni singola impostazione, dei valori predefiniti e degli intervalli validi, consulta la [Guida di Riferimento alle Impostazioni](../reference/settings.md).
