# Settings Page

The Settings page is accessible from any screen by tapping the **gear icon** in the top-right corner of the app bar. A **search icon** in the top bar lets you jump directly to a section by name.

Settings are grouped into the following sections:

| Section | What it controls |
|---|---|
| **Discover** | Multicast address and port, unicast fallback scan, discovery and connection timeouts |
| **Project** | Whether to load communication objects; number of items shown per page in the project tree |
| **Monitor** | Keep-screen-on during monitoring; maximum telegram buffer size |
| **Scan** | Programming mode scan duration and ping interval (used by the Management → Prog. Mode tab) |
| **Subscription** | Current plan status, restore purchases, cancel subscription, Privacy Policy, Terms of Service |
| **Language** | App interface language (English, German, French, Spanish, Italian) — takes effect immediately |
| **Theme** | Light, Dark, or Match system |
| **Send Feedback** | Leave a review, report a bug, write to us |

For the full description of every individual setting, default values, and valid ranges, see the [Settings Reference](../reference/settings.md).
