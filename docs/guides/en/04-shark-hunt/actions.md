# Actions Reference

A Shark Hunt action is a single executable item inside a hunt — either a command sent to the KNX bus or a bus monitor with a specific filter. This guide covers all available action types and their configuration options.

Actions are created during [Step 3 of the hunt creation wizard](create-hunt.md#step-3--actions) and can be edited later via the hunt editor.
